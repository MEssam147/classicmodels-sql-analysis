# ClassicModels SQL Analytics Project

This project demonstrates advanced SQL analysis using the ClassicModels relational database (MySQL).

---

##  Database Information
- Database: ClassicModels
- SQL Dialect: MySQL
- Focus: Sales analytics & customer behavior analysis

---

##  Skills Applied

- Multi-table INNER JOIN and LEFT JOIN
- CTEs (Common Table Expressions)
- Window Functions (ROW_NUMBER, LAG)
- CASE statements for customer segmentation
- Aggregations (SUM, COUNT)
- Date functions (DATEDIFF, DATE_ADD, DATE_SUB)
- Customer purchase behavior and sales change analysis

---

##  Business Questions Answered

1. What is the total sales value per order?
2. How does customer purchase value change over time?
3. How can customers be segmented by credit limit?
4. What are sales trends across product lines and regions?
5. How can repeat purchases be identified using window functions?

---

##  Project Structure

- `queries.sql` → Contains structured and optimized SQL queries
- `SQL Queries and Results/` → Query result screenshots

---

## 🎯 Objective

This project focuses on writing optimized, structured, and analytical SQL queries to extract meaningful business insights from a relational database.
